Exemplo 13 - CSS Bootstrap

O bootstrap é um framework front-end popular para desenvolvimento de aplicações web. Ele foi criado pelo antigo Twitter com a finalidade de facilitar a criação de páginas. Eles possui um conjunto de seletores de classe que podem ser aplicadas as mais diversas situações de estilização de um documento HTML.
