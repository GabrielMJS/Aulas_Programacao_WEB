Exemplo 9 - CSS

O CSS é utilizado na estilização de elementos HTML. Para que você possa aplicar a folha de estilo em cascata - CSS nos elementos HTML,você necessariamente poderá utilizar as regras CSS que podem se definidas por meio de seletores seguidos de {} e dentro dessas chaves propriedades e valores.

Os seletores contidos nas regras CSS, são padrôes usados para definir quais elementos HTML receberão determinados estilos. Eles podem ser de três tipos: Seletores de Tipo ID e Classes.

Seletores de Tipo

Os seletores de tipo são aqueles que são representados pelas Tags HTML, geralmente são aplicados a todas as tags HTML que se referenciam.

Esses Seletores são descritos da seguinte forma: h1 {propriedade:valor;}, no caso de exemplo citado o seletor de tipo é o h1, que representa uma tag de seção.

Seletores de ID

Os seletores de id são aqueles que utilizam id (atributos) únicos de elementos HTML. Esses seletores são personalissimos, pois são aplicados nicamente aos elementos HTML que tiverem o ID específico.

Para criar um seletor de ID seguimos a seguinte sintaxe: #NomeSeletor{propriedade:valor}

Seletores de Classe

Os seletores de classe são aqueles usados para aplicar o estilo a vários elementos HTML que possuírem o mesmo atributo class. Sua sintaxe é definida da seguinte maneira: .NomeSeletor{propriedade:valor}
