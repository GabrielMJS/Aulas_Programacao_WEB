O que é o Grid Layout

O Grid layout é uma técnica ára contrução de layouts mais complexos, o seu layout é basedo em linhas e colunas , ou seja ele é bidimensional.

Quando comparado ao flexbox nos temos algumas diferenças, pois o flexbox é unidimensional, ordenando os elementos em linha.

Flexgrid ou Gridlayout

O flexbox e o Grid Layout podem ser usados em conjunto. Voçê pode aplicar o flexbox a elementos dentro de uma layout de grad, aproveitando o melhors das técnicas.

A escolha entre flexgrid e o Grid Layout não se trata de um substituir o outro, mas sim de entender as necessidades especificas do seu layout e escolher a técnica que melhor as atende.
