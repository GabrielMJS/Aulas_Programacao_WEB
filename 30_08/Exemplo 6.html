Web Semântica

O termo web semântica se refere a um conjunto de conceitos que estão interligados e que foi inicialmente cunhado pelo consorcio W3C.Essencialmente, ele diz respeito ás diretrizes para tornar os armazenamentos de documentos HTML mais interoperáveis.

A web semântica é um movimento colaborativo para organizer a informação de forma mais legivel para a máquina, por meio de padrões de formtação de dados
